    Delete / Scope Review

    Exclude paper from site?

    This excludes the paper from public preview and map exports. It does not directly edit OpenAlex-derived processed CSV files. A durable exclusion record prevents the paper from reappearing in future exports.

    Restoring preserves the exclusion audit record but marks it inactive. Run the public preview exporter afterward to return an otherwise eligible paper to the site.
